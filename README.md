# my_portfolio_dev

Professional portfolio for **Md Tahsin** — AI Engineer & Researcher (MagicMind, EWU). Built with React, TypeScript, Vite, and Tailwind CSS.

## Live demo

Deployed on [Vercel](https://vercel.com) from this repository.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```

## Stack

- React 19 + TypeScript + Vite
- Tailwind CSS
- Swiper (hero slider + publications carousel)
